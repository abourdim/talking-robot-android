# 🤖 Talking Robot — Android wrapper

Capacitor wrapper that packages the [talking-robot](https://github.com/abourdim/talking-robot) web app as an installable **Android APK**.

> الروبوت الناطق · TALKING ROBOT

Interactive web app that pairs with a BBC micro:bit V2 over Bluetooth Low Energy (BLE). Part of the workshop-diy.org educational series for kids.

## Quick start

```bash
./manage.sh    # auto-detects your OS, then opens the menu
```

Build, test on emulator, and prepare a Play Store release from a single guided menu. See [WORKSHOP.html](WORKSHOP.html) for the full step-by-step guide.

## Download the latest APK

[📦 v1.0-debug release](https://github.com/abourdim/talking-robot-android/releases/latest) · [![QR](releases/download-qr.svg)](https://github.com/abourdim/talking-robot-android/releases/latest)

Scan the QR with an Android phone, or `adb install -r talking-robot-v1.0-debug.apk`.

## Capabilities required at runtime

- Bluetooth Low Energy (Web Bluetooth → @capacitor-community/bluetooth-le)
